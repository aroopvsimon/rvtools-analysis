# RVTools Analysis Starter

Static GitHub Pages app for parsing RVTools Excel exports in the browser.

## Files
- `index.html`
- `styles.css`
- `script.js`

## GitHub Pages
After uploading these files to your repo:
1. Go to **Settings** → **Pages**
2. Under **Build and deployment**, choose **Deploy from a branch**
3. Branch: **main**
4. Folder: **/ (root)**
5. Save

Your page will go live at:
`https://aroopvsimon.github.io/rvtools-analysis/`
